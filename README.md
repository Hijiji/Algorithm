# 구간합 공식
**합배열공식**
 - S[i] = S[i-1] + A[i]

**구간합공식**
- S[j] - S[i-1]

**2차원 구간합배열 만드는 공식**
- D[i][j] = D[i][j-1] +D [i-1][j] - D[i-1][j-1] + A[i][j]

**특정 구간합 계산 공식**
- D[x1][y2] - D[x1-1][y2] - D[x2][y1-1] + D[x1-1][y1-1]
